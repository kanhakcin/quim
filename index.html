🚀 Bem-vindo ao Baralho Cósmico Atômico! 🌌

🌀 Selecione cartas para formar ligações químicas interestelares!

🧪 Regras do Jogo

🎯 Objetivo

Forme ligações químicas válidas para ganhar pontos! O jogador com mais pontos quando o baralho acabar vence!

⚗️ Ligação Iônica (Metal + Ametal)

Soma das valências deve ser par • 25 + (nº ametais × 5) pontos
Ex: Ca(v=2) + 2Cl(v=1) = 4 (válida!)

🔗 Ligação Metálica (2+ Metais)

20 pontos fixos

🔬 Ligação Covalente (2+ Ametais)

20 + (nº cartas × 10) pontos

⭐ Bônus

  • Mão Limpa: Jogue todas 5 cartas = ×1.5 pontos
  • Cartas Especiais: Break Bond, Double Points, Catalyst
  • Gases Nobres: Bloqueiam ligações

🎮 Como Jogar

  1. ❓ Clique para regras
  2. 🚀 "Começar Jogo" na tela inicial
  3. 1 carta = Comprar/Nova carta
  4. 2+ cartas = Confirmar ligação
  5. Deck acaba = FIM! Vencedor anunciado
